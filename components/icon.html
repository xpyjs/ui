Icon 图标 ​

提供各种图标显示的组件，支持 iconify 图标库和自定义 SVG 内容。

使用指南 ​

Iconify 图标 ​

Icon 组件默认支持 Iconify 图标库，这是一个包含超过 150,000 个开源图标的集合。您可以在 Iconify Icon Sets 浏览并查找所需的图标。图标名称格式为 {集合名称}:{图标名}，例如 mdi:home、fa:user 等。

参数 ​

Props ​

名称类型默认值描述
namestring''图标名称，支持 iconify。也支持以 url: 前缀传递地址
contentstring''一段 svg 图标文本内容。优先级比 name 高，会覆盖 name
colorstring'currentColor'图标颜色
sizestring | number'1em'图标大小
